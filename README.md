# שומר עברית — TypeBeep

לינק להורדה: https://github.com/devsite350-br/type-beep/raw/main/TypeBeep-Setup.exe

תוכנת Windows קטנה שרצה ליד השעון ומתריעה בצליל עדין (או הבהוב מסך) כאשר:
- **CapsLock דלוק**
- **המקלדת לא בעברית**

## הפעלה
לחיצה כפולה על `TypeBeep.exe`. התוכנה נרשמת אוטומטית לעליית המחשב (ניתן לביטול בהגדרות).

## תכונות
- צלילים עדינים שנוצרים בקוד (פעמון, צליל כפול, טיפה, נצנוץ) — בחירת צליל נפרד לכל התראה + עוצמה
- מצבי התראה: צליל בלבד / צליל + הבהוב מסך / הבהוב מסך בלבד
- "התרע רק כשמתחילים להקליד" — מונע התראות שווא כשמחליפים שפה בכוונה
- תזכורת חוזרת כל 30 שניות כש-CapsLock נשאר דלוק (ניתן לכיבוי)
- רשימת תוכנות מוחרגות (למשל תוכנות שבהן מקלידים אנגלית בכוונה)
- השהיה לשעה מהכפתור/מהאייקון, או Ctrl+Alt+H להשהיה/הפעלה מיידית
- **המרת טקסט לעברית (Ctrl+Alt+H)**: הקלדת בטעות באנגלית (למשל "cuer yuc")? לחיצה אחת מוחקת
  את מה שהקלדת ומקלידה במקומו את העברית הנכונה ("בוקר טוב") — בלי צורך לסמן. עובד גם על
  טקסט מסומן ידנית. אחרי ההמרה המקלדת עוברת אוטומטית לעברית. הקיצור ניתן לשינוי בהגדרות,
  וזמין גם מתפריט קליק ימני על האייקון
- אייקון ליד השעון מחליף צבע: ירוק = תקין, אדום = לא עברית, כתום = CapsLock, אפור = מושהה
- ההגדרות נשמרות ב-`%APPDATA%\TypeBeep\settings.txt`

## יציאה מלאה
קליק ימני על האייקון העגול ליד השעון ← "יציאה" (סגירת החלון רק מסתירה אותו).

## בנייה מחדש מהמקור
```
build.cmd
```
(משתמש בקומפיילר C# המובנה ב-Windows — אין צורך בהתקנות)
